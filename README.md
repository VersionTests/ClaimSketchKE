# ClaimSketch — Landing Page

This repository contains a responsive, dark-themed landing page for ClaimSketch — an AI-powered insurance claim submission portal. The design uses a blue-accent, secure-looking theme and incorporates your logo and app screen preview.

Files
- index.html — main landing page
- styles.css — styling
- assets/logo.png — place your ClaimSketch logo image here (image 2)
- assets/screen.png — place the app/screen preview image here (image 1)

Replace images
- Put the images you uploaded into the assets/ folder and name them:
  - assets/logo.png  (the ClaimSketch logo)
  - assets/screen.png (the tall mobile screen mock)

Run locally
1. Clone the repo:
   git clone https://github.com/MUNJINGUGI/claimsketchKE.git
2. Open index.html in a browser.

Deploy to GitHub Pages
- Option A (serve from main branch root):
  1. Push files to your repository's default branch (main).
  2. In GitHub repo Settings → Pages, set "Branch: main / (root)" and save.
  3. Once the repo is public, the site will be available at:
     https://MUNJINGUGI.github.io/claimsketchKE/

- Option B (if you prefer gh-pages branch):
  1. Create a branch named gh-pages and push files there.
  2. In Settings → Pages select gh-pages branch.

Form
- The contact form currently uses a Formspree placeholder action:
  https://formspree.io/f/your-form-id
  Replace with your own endpoint if needed.
